# Core Engine Scaffold

This folder contains the first v0.3 core modules.

## Modules

- scene/schema.js: scene contract validator.
- eval/evaluate-graph.js: deterministic graph -> scene evaluator.
- index.js: public exports for core API.

## Current scope

This is a scaffold for Phase 1 and not yet wired into live preview/export paths.
It establishes a stable scene contract so all adapters can converge later.
